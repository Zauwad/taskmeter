Question-1: What are the different ways to select an element in the DOM?

-------------------------------------------------------------------------------

You can select elements in the DOM using methods like getElementById() for an ID, getElementsByClassName() for a class, getElementsByTagName() for a tag, and querySelector() or querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

-------------------------------------------------------------------------------

innerHTML retrieves or sets the HTML content inside an element, including tags, while innerText gets or sets the visible text, excluding any HTML tags

Question-3: What is event delegation in the DOM?

-------------------------------------------------------------------------------

Event delegation is a technique where you attach a single event listener to a parent element instead of adding individual listeners to each child element

Question-4: What is event bubbling in the DOM?

-------------------------------------------------------------------------------

Event bubbling is a concept in the DOM where an event starts from the target element and then bubbles up to its parent elements in the DOM hierarchy.

Question-5: How do you create, add, and remove elements using JavaScript?

-------------------------------------------------------------------------------

To create an element, use document.createElement() and set its properties. To add it to the DOM, append it to a parent element, and to remove it, call remove() on the element.
